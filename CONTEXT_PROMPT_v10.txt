## 📝 FICHIERS CRITIQUES (CONFIGURATION FINALE)

### ~/.config/ART/options - Section [External Editor]
```
[External Editor]
EditorKind=1
GimpDir=
PhotoshopDir=
CustomEditor=
OutputDir=0
CustomOutputDir=
Float32=false
BypassOutputProfile=false
```
⚠️ **CRITIQUE:** EditorKind DOIT être 1, CustomEditor DOIT être VIDE!

### ~/.config/ART/usercommands/select-editor.txt
```
[ART UserCommand]
Label=Le ARTherapee Compagnon
Command=/home/carafife/art-compagnon/select-editor.py
FileType=raw|jpg|jpeg|tif|tiff|png
NumArgs=1
```

### ~/.config/ART/usercommands/smart_masking.txt
```
[ART UserCommand]
Label=AI masking (SMART)
Command=./bash/smart_masking.sh
FileType=jpg|jpeg|tif|tiff|png
NumArgs=1
```

### ~/.config/ART/usercommands/nind_denoise_raw.txt
```
[ART UserCommand]
Label=Synde Noise (nind-denoise)
Command=./bash/nind_denoise_raw.sh
FileType=raw
NumArgs=1
```
