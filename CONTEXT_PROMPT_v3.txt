## ✅ SOLUTIONS CRITIQUES

### 1. CTL FIX (ART compilation)
```bash
cmake .. -DENABLE_CTL=ON -DCTL_INCLUDE_DIR=/usr/include/CTL
```

### 2. AC FIX (NumArgs usercommand)
```
NumArgs=1
```
(IMPORTANT: Dans ~/.config/ART/usercommands/select-editor.txt)

### 3. /TMP FIX (Espace disque)
```bash
rm -rf /tmp/*.tif /tmp/*.TIF
```

### 4. .ARP CACHE FIX (Profils corrompus)
```bash
rm -rf ~/.cache/ART/profiles/
```

## 🛠️ TECHNOLOGIES EMPLOYÉES

### ARTcompagnon (AC)
- **Language:** Python 3 + PyQt5
- **Support scripts:** Bash, Python 3, Lua
- **UI:** QTreeWidget (arborescence), gnome-terminal (exécution)
- **Config:** ~/.config/ART/usercommands/select-editor.txt

### SMART (SAM2 Mask Builder)
- **Format:** JPG/TIF/PNG UNIQUEMENT (pas RAW!)
- **Device:** device=cpu (AMD Radeon)
- **Model:** sam2.1_hiera_base_plus.pt
- **Config:** ~/.config/ART/SMART.json

### nind-denoise (AI Denoiser)
- **Format:** RAW UNIQUEMENT
- **Device:** cpu (AMD Radeon)
- **Output:** -denoised.tif

### ART CTL Scripts
- **Dépendances Fedora:** ctl, CTL-devel, openexr, openexr-devel
- **ART compilation:** -DENABLE_CTL=ON
- **Dossier:** ~/.config/ART/ctlscripts/

## 🚨 POINTS D'ATTENTION IMPORTANTS

1. **Repo correct:** ~/art-compagnon/ (minuscule, pas ~/ARTcompagnon/)
2. **SMART:** Exporter RAW → JPG avant de l'utiliser
3. **CTL:** Recompiler ART avec ENABLE_CTL=ON
4. **GPU:** AMD Radeon = device=cpu (JAMAIS CUDA)
5. **AC vs usercommands:** AC = depuis éditeur ART | usercommands = depuis navigateur fichiers
6. **Nettoyage:** /tmp/*.tif + ~/.cache/ART/profiles/ régulièrement
